# Project Access and Links

## 1. Local Development
To see the website on your computer:
1. Open your terminal.
2. Run the command: `npm run dev`
3. Open your browser and go to: [http://localhost:3000](http://localhost:3000)

## 2. Adding links in code
If you want to add a link to a button or text in your project, use the Next.js `Link` component:

```tsx
import Link from 'next/link';

<Link href="/your-path">
  Click here
</Link>
```

## 3. Live Website (GitHub Pages)
If you have deployed the site, the link is usually:
`https://yourusername.github.io/dagalti/`
*(Note: I couldn't find a remote URL set up in your local Git config yet).*
